# Personal website

### Info
Personal website, written in Svelte.

>[!IMPORTANT]
> Yes, I know I used a clanker for this project but it was fixing 3 little bugs I swear :(

### Quickstart
You must have:
* Bun

#### Steps
1. Clone this repo
```bash
   git clone https://github.com/Snowflake6413/sveltesite
```

2. Enter directory
```bash
cd sveltesite
```

3. Install dependencies
```bash
bun install
```
4. Run the server
```bash
bun run dev
```
5. Profit
